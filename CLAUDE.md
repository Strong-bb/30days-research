# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 项目概述

这是一个个人研究文档项目，用于收集和整理过去 30 天内的技术话题深度研究报告。所有文档使用中文撰写。

## 仓库结构

- 研究报告以 Markdown 文件存储，文件名使用中文或英文均可，正文必须使用中文
- 报告通常由 `/last30days` 技能生成后整理而成
- 每份报告包含：话题背景、核心观点、数据来源、分析判断

## 文档规范

- 文件名：使用能体现话题的简短标识，如 `karpathy-llm-知识库研究.md`
- 正文语言：**中文**
- 格式：标准 Markdown，支持表格、引用、代码块
- 每份报告应包含生成日期和数据来源

## 研究工具

- 使用 `/last30days` 技能进行多平台深度研究
- 可用 Hacker News API (`hn.algolia.com`) 作为免费数据源补充
- 网页内容可通过 `curl` 直接抓取补充细节
- 每次研究完毕 都要落文档
